# Capability Freeze Criteria

Freezing a capability is a governance decision.
A capability MUST NOT be frozen without the following evidence.

## Required Evidence

- Canonical mutating function template enforced
- Deterministic execution tests passing
- Replay protection tests passing
- Audit logging tests passing
- Negative tests present
- Public surface enumerated and reviewed
- Composition risk assessment written

## Governance Rule

Freezing without documented evidence is a STOP CONDITION.
